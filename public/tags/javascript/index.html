snap面试

下午去豌豆实验室面试了他们家的前端工程师的职位，这应该是除了当时面试实习生后的第一次前端方面的面试，虽然十有八九都挂了，但有些东西还是值得记录一下的。

先说一下关于这次面试的体验吧，一共经历了笔试和三个人的面试，面试官对新的东西都了解的比较多，其中有一个哥们儿不是很喜欢。一副看什么都很easy的样子。
总之这次面试还是暴露出自己不少的问题：

  • 对事件的理解不够透彻
  • CSS3相关属性的优缺点掌握不牢靠
  • HTML5只是部分了解，但是还是缺少深入的了解，居然把标签名记错了
